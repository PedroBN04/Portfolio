Projeto de Análise de Dados Agrícolas

Este projeto foca na análise de dados agrícolas utilizando Python, SQL e Power BI. Ele envolve a extração, limpeza, armazenamento e visualização de dados para identificar padrões produtivos e otimizar processos agrícolas.

Principais etapas:
